The protocol is reversible.
The codec is not semantic.
The browser does all encoding locally.
The worker keeps the page responsive.
The dictionary is a static asset.
The payload is shorter only when zstd wins.
Raw is kept when compression is not shorter.
Short messages often stay raw.
Longer mixed messages can benefit from the dictionary.
URLs often repeat https, query, mode, lang, and codec.
UI messages often repeat ready, loading, failed, input, output, copy, clear.
Chinese chat messages often repeat 请、我想、可以、不要、正在、失败、成功。
Emoji and symbols must round-trip unchanged: 😺 🧶 🍣 ✨ ✅ ❌ ⚠️。
Fullwidth punctuation must round-trip unchanged: ！ ～ ， 。 ： ； （ ） 【 】。
ASCII punctuation must round-trip unchanged: ! ~ ? # @ / \ & = % + - _ . : ;
Markdown must round-trip unchanged: **bold** `code` [link](https://example.com)
JSON must round-trip unchanged: {"ready":true,"codec":"zstd-dict"}
Shell commands must round-trip unchanged: pnpm test && pnpm build
Query strings must round-trip unchanged: ?mode=encode&lang=zh-CN&copy=1
Hash fragments must round-trip unchanged: #result
Paths must round-trip unchanged: /src/assets/dict_v1.bin
